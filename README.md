# 简介

PKD 是一个期末作业项目。

主要有三部分组成。
- 计算器
- 学生信息管理系统
- 优惠券模块

本系统可实现学生信息的注册、查询、删除、修改、退出系统功能。除此之外制作了一个简易计算器，计算器的使用需要输入优惠券，优惠券可通过学生信息管理系统获取。

# 快速入门
1. pip install -r requirements.txt
2. python init_db.py
3. python -m app run
